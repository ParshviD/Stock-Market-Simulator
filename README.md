# 📈 Stock Market Simulator

A Python-based stock market simulator that allows users to simulate buying and selling stocks, manage a portfolio, and visualize stock trends.

## 📌 Overview

This project simulates a virtual trading environment where users can interact with predefined stocks using an initial balance of ₹10,000. It helps in understanding how stock transactions and portfolio management work without real financial risk.

## ⚙️ Features

* Buy and sell stocks dynamically
* Track portfolio balance and holdings
* Calculate profit/loss
* Compare stock trends using graphs (Matplotlib)
* Interactive command-line interface

## 🧠 Concepts Used

* Python programming
* Dictionaries and data structures
* Functions and modular design
* Data visualization using Matplotlib

## 🔧 Working

* Users start with ₹10,000 virtual balance
* Choose from predefined stocks
* Perform buy/sell operations
* Portfolio updates in real time
* Graphs simulate stock trends using variations

## 📊 Scope

* Virtual trading simulation
* Portfolio management
* Graph-based stock comparison

## ⚠️ Limitations

* Uses predefined stock prices (no real-time data)
* Limited number of stocks
* Simplified market behavior

## 🚀 Future Improvements

* Integration of real-time stock data
* Addition of more stocks
* Advanced analytics and visualization

## ▶️ How to Run

1. Install required library:
   ```
   pip install matplotlib
   ```
2. Run:
   ```
   python stock_simulator.py
   ```
